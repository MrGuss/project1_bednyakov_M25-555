# Лабиринт сокровищ

Лабиринт сокровищ - это текстовая игра, в которой игрок находится в лабиринте и должен найти выход, решая загадки и собирая предметы.

## Установка

Требования:
- Python 3.8+
- Poetry

```bash
make install
```

## Запуск

```bash
make project
```
